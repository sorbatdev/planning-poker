# Planning Poker

[Go to App](https://sorbatdev.github.io/planning-poker/)

A little real-time planning poker app, built mostly for fun and kept running so old colleagues (and anyone else who wants it) always have a free, no-login place to run estimation sessions together.

Create a room, share the ID, and vote with configurable granularity decks — standard, half-point, or your own custom set

Built with plain HTML/CSS/JS and Firebase Realtime Database, deployed via GitHub Pages. It's a hobby project maintained on and off.

Do not abuse.
